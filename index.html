💡 Quick Start

Add one line to your <head>:

<link rel="stylesheet"
  href="style.css"/>

Plain HTML is styled automatically. Reach for the class helpers when you need a specific component.

📢 About this stylesheet [Manage]
★ Design goals
Capture the look of early-2000s corporate web UI — the SharePoint task panes, Outlook Today dashboards, beveled Windows XP chrome, and Tahoma-heavy intranet portals. Styled base elements let plain HTML work out of the box; semantic classes let you compose richer components.
ℹ How to use
Each section below shows a component, the class name, and where to reach for it. Copy the markup patterns directly — the CSS file is unminified and commented so you can grep for /* 13. Buttons */-style section headers.
✓ Accessibility notes
Focus outlines are preserved on links (dotted navy). Custom checkboxes use accent-color. Respect aria-current="page" for tabs and nav items. Colour is never the only signal.
